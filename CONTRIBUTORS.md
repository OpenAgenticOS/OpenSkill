# Contributors · 贡献者

**中文：** GitHub **[Contributors 图谱](https://github.com/OpenAgenticOS/OpenSkill/graphs/contributors)** 根据**已合并提交**自动生成。若未出现你的头像，请检查 `git config user.email` 是否与 GitHub 已验证邮箱一致。

**English:** The **[GitHub contributors graph](https://github.com/OpenAgenticOS/OpenSkill/graphs/contributors)** is generated from **merged commits**. If you are missing, ensure `git config user.email` matches a verified address on GitHub.

在技能 frontmatter 中填写 `author`（GitHub 用户名）可署名。Use the `author` field in each skill’s frontmatter for credit.
