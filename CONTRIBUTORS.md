# Contributors · 贡献者

OpenSkill 感谢每一位贡献者。完整提交记录见：

- [GitHub Contributors 图谱](https://github.com/OpenAgenticOS/OpenSkill/graphs/contributors)

在 Skill 文件的 YAML frontmatter 中填写 `author`（GitHub 用户名）可在技能元数据中署名。若你希望在本文件列出姓名，可向维护者提交 PR 补充。

---

OpenSkill thanks everyone who contributes. See the [GitHub contributors graph](https://github.com/OpenAgenticOS/OpenSkill/graphs/contributors) for the full list.

Use the `author` field in each skill’s frontmatter to credit your GitHub username.
