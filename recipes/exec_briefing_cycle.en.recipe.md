## Scenario

Compress long inputs for leadership or boards using **executive summary, one-pager decision memo, deck outline, or data storytelling**—mix and match. Roles are typical consumers; anyone briefing upward can use these.

## Suggested path (optional)

1. **Long reads:** `executive_summary`.
2. **Explicit decision ask:** `decision_memo_one_pager`.
3. **Live meeting:** `presentation_outline` then build slides yourself.
4. **Metric-heavy:** `data_storytelling` for narrative + chart guidance.

## Note

Not legal/compliance advice—always have humans review before submission.
